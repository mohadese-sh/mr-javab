# Icons Directory

This directory contains shared icon files used throughout the website.

## Usage

Place all icon files (SVG, PNG, ICO, etc.) in this directory.

## Icon Types

- **SVG icons** - Scalable vector icons (recommended)
- **PNG icons** - Raster icons for specific sizes
- **Icon fonts** - Font-based icon sets
- **Favicon** - Site favicon files

## Recommended Structure

- `svg/` - SVG icon files
- `png/` - PNG icon files
- `favicon/` - Favicon files

## Usage in HTML

```html
<!-- SVG Icon -->
<img src="icons/svg/icon-name.svg" alt="Icon">

<!-- Or inline SVG -->
<svg><use href="icons/svg/icon-name.svg#icon"></use></svg>
```

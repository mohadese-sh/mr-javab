# Yekan Bakh FaNum Font Files

Place your Yekan Bakh FaNum font files in this directory.

## Required Font Files:

- `YekanBakhFaNum-Regular.woff2` (or `.woff`, `.ttf`)
- `YekanBakhFaNum-Bold.woff2` (or `.woff`, `.ttf`)
- `YekanBakhFaNum-Heavy.woff2` (or `.woff`, `.ttf`)

## Where to Download:

You can download Yekan Bakh FaNum font from:
- https://fontchi.com/fonts/yekan-bakh/
- https://fontfont.ir/
- Or other Persian font repositories

## Font Weights:

- Regular (400): For normal text
- Bold (700): For bold text
- Heavy (900): For headings and emphasis

After placing the font files in this directory, the website will automatically use them.
